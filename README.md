### [Практическая работа №3 — Планирование релизов и SLA](Практическая работа_3.pdf)
Темы:
- формирование roadmap релизов;

- работа со SLA;
- анализ рисков и «чёрных лебедей»;
- постановка целей и определение зависимостей.

### [Практическая работа №4 — Передача требований и SLA](Практическая работа_4.pdf)
Темы:
- схема взаимодействий команд (PO → TL → Dev/QA → Support);
- документ передачи требований (PRD) и SLA;
- анализ проблем коммуникации;
- корректирующие действия.

### [Практическая работа №5 — Trade-off и метрики релиза](Практическая работа_5.pdf)
Темы:
- приоритизация задач (Value/Effort, WSJF);
- расчёт TCO, CPU, ROI, Run vs Grow;
- влияние «чёрного лебедя»;
- пересчёт приоритетов;
- рекомендации для релизов X+N.

## Используемый кейс
Все работы построены вокруг развития функциональности интернет-магазина:
- поиск и корзина;
- модуль доставки (слоты, ETA, трекинг);
- поддержка SLA и механизмов мониторинга;
- развитие продукта в релизном цикле X → X+N.
